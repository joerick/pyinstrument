## 问题与修复记录

### 跨平台路径显示差异

问题：

- 在 Linux CI 中，`group_description()` 产生的库名与 Windows 本地环境不同。

修复：

- 将断言从写死 `env` 改为更稳健的跨平台匹配策略。

### Session.frame_records 类型检查问题

问题：

- 测试中构造的 `frame_records` 不符合 `FrameRecordType` 定义。

修复：

- 调整为符合类型定义的结构，并使用类型转换明确表达测试意图。

### CLI 目标描述测试问题

问题：

- `test_target_description_format` 使用 `str(bytes)` 做断言，导致 Windows 路径转义后匹配失败。

修复：

- 改为 `decode("utf-8", errors="replace")` 后再断言。

### pre-commit 格式化失败

问题：

- `black`、`isort`、`end-of-file-fixer` 在 CI 中自动改动文件，导致工作流失败。

修复：

- 本地执行等价格式化并提交结果。
